# SleepApneaBD — Sleep Apnea & Breathing Clinic (DSRC Dhaka)

Modern static website for Bangladesh's first sleep apnea clinic.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage with all clinic sections |
| `sleep-apnea-assessment.html` | STOP-BANG sleep apnea risk screener |
| `articles.html` | Articles & resources grid |
| `contact.html` | Contact form + clinic info |

## Tech Stack

Pure static HTML + CSS + JS — no build tools, no frameworks.  
Hosted on **GitHub Pages** at https://sleepapneabd.com

## Design

- Dark glassmorphism theme with frosted glass cards
- Gradient accents: green (#06d6a0) → blue (#3b82f6) → purple (#8b5cf6)
- Dark/Light mode toggle
- Fully responsive (mobile-first)
- Bengali + English bilingual content
